# 演習: RAG ミニプロジェクト

1) 埋め込みを作成して検索する
- `rag_demo.ipynb` を使用して、与えられたサンプルテキスト群から埋め込みを作成し、クエリに対して上位 3 件を返す関数を実装してください。

2) LLM との統合（設計）
- 検索結果をプロンプト化して LLM に渡す設計を考え、短いプロンプトテンプレートを作成してください（200 トークン以内推奨）。

3) Faiss でスケールさせる
- `scripts/rag_faiss_example.py` を実行して、Faiss インデックスの作成と検索を確認してください。Faiss が利用できない場合はノートブック内の簡易検索に戻ってください。

発展課題:
- 実際のドキュメント群（複数テキストファイル）を読み込み、チャンク化してインデックス化、検索精度を評価する小さな実験を行ってください。
