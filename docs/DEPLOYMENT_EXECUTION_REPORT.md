## 💾 バックアップ・リカバリ計画

### デプロイ前バックアップ

```bash
# イメージバックアップ
docker save project_root-streamlit-ui:latest > \
  backup_image_$(date +%Y%m%d_%H%M%S).tar

# 設定バックアップ
cp -r config/ config.backup/
cp -r corpus/ corpus.backup/

# チェックポイントバックアップ
cp -r checkpoints/ checkpoints.backup/
```

### ロールバック手順

```bash
# 1. 新しいコンテナを停止
docker-compose -f docker-compose.production.yml down

# 2. 前のイメージを復元
docker load < backup_image_$(最新).tar

# 3. コンテナ再起動
docker-compose -f docker-compose.production.yml up -d

# 4. ヘルスチェック
curl http://localhost:8501/healthz
```
