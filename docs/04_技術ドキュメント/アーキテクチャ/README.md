# アーキテクチャ設計ガイド（テンプレート）

## 概要
本ガイドはシステム全体のアーキテクチャ設計方針・構成・設計図をまとめるためのテンプレートです。

## 主要構成・フロー
- システム全体図（Mermaid/画像推奨）
- 各レイヤ・モジュールの役割
- データフロー・依存関係

## サンプル構成図
```mermaid
graph TD
    UI --> API
    API --> LLM
    LLM --> DB
    API --> 外部サービス
```
