## スタート地点別ナビゲーション

### 🎯 シナリオ別アクセスパス

#### シナリオ 1: プロジェクトの全体像を理解したい

```
1️⃣ README.md を読む (3分)
   ↓
2️⃣ docs/00_プロジェクト概要/統合最終レポート.md を読む (15分)
   ↓ 
3️⃣ docs/03_Phase7-10資料/技術アーキテクチャ.md で設計を確認 (10分)
   ↓
4️⃣ docs/04_完了レポート/Phase別レポートで詳細確認 (20分)

所要時間: 約50分
理解度: 初級 ↓ 上級 (全体像把握)
```

#### シナリオ 2: セキュリティ機能について学びたい

```
1️⃣ src/security_core/ ディレクトリ確認

   ↓

   ↓
3️⃣ tests/test_phase10_auth.py でテストケース確認
   ↓
4️⃣ Phase10 認証実装を詳細確認
   - src/phase10/next_gen_auth.py
   - src/phase10/next_gen_auth_components.py

所要時間: 約2時間
理解度: 中級 ↓ 上級 (セキュリティ機構把握)

#### シナリオ 3: コード開発・修正に取り組みたい

```
1️⃣ docs/01_クイックスタート/セットアップガイド.md で環境構築
   ↓

   ↓
3️⃣ src/phase10/ の該当実装を確認
   ↓
4️⃣ tests/test_phase10_*.py で テストを実行
   ↓
5️⃣ 修正コミット前に
   - pytest tests/test_phase10_*.py -v で全テスト確認
   - コードレビュー基準を docs/技術ドキュメント/運用ガイド.md 確認

所要時間: 変動 (開発スコープに依存)
必須チェック: テスト 100% PASS, コードレビュー合格

#### シナリオ 4: 本番展開・運用サポート

   ↓

5️⃣ デプロイ実行

所要時間: 約2時間 (初回) or 30分 (定期)
必須知識: 運用ガイド必読, DR手順暗記

## 実行・テストコマンド

### 環境セットアップ

```bash
# 仮想環境構築
python -m venv .venv
source .venv/bin/activate    # Linux/Mac
.venv\Scripts\activate        # Windows

# 依存パッケージ インストール
pip install -r requirements.txt

# テスト環境検証
pytest tests/test_phase7_soc.py -v  # 必須 PASS 確認
```

### テスト実行

```bash
# Phase別テスト
pytest tests/test_phase7_soc.py -v              # 60 テスト
pytest tests/test_phase8_detection.py -v        # 50 テスト
pytest tests/test_phase9_integration.py -v      # 80 テスト
pytest tests/test_phase10_soc.py -v             # 25 テスト

# Phase 10 全テスト (現在 67/117 成功)
pytest tests/test_phase10*.py -v

# 全テスト実行
pytest tests/ -v

# テスト結果サマリー
pytest tests/ -v --tb=no | tail -20
```

### パフォーマンス測定

```bash
# 実行時間計測
pytest tests/test_phase10_soc.py -v --durations=10

# Memory プロファイリング
python -m memory_profiler src/phase10/threat_detection.py

# CPU プロファイリング
python -m cProfile -s cumulative src/phase10/threat_detection.py
```

### デプロイ検証

```bash
# 本番前 チェックリスト
python tools/deployment_validator.py

# Docker イメージビルド
docker build -t phase10-soc:prod .

# ローカル検証
docker run -it phase10-soc:prod /bin/bash
pytest tests/test_phase10_soc.py -v
```
