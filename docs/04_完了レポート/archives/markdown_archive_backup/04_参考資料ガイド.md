## 📚 必読の原論文

### Tier 1: 必須

#### 1. "Attention Is All You Need" (2017)
- **著者**: Vaswani et al.
- **学会**: NeurIPS 2017
- **URL**: https://arxiv.org/abs/1706.03762
- **重要度**: ⭐⭐⭐⭐⭐ (5/5)

**内容**:
- Transformer アーキテクチャ初提案
- Multi-Head Self-Attention
- Position-wise Feed-Forward Network
- 位置エンコーディング

**読むべき理由**:
- Attention メカニズムの基礎
- 現代の NLP モデルの土台
- わかりやすい図解が豊富

**読み方**:
```
第1回: Section 1-2 (導入と概要) - 30分
第2回: Section 3 (モデルアーキテクチャ) - 1時間
第3回: Section 4 (実験結果) - 30分
第4回: 全体読み返し + ノート作成 - 1時間
```

#### 2. "BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding" (2018)
- **著者**: Devlin et al.
- **学会**: ICLR 2019
- **URL**: https://arxiv.org/abs/1810.04805
- **重要度**: ⭐⭐⭐⭐⭐

**内容**:
- Bidirectional Transformer (BERT)
- Masked Language Model (MLM)
- Next Sentence Prediction (NSP)
- Pre-training と Fine-tuning

**読むべき理由**:
- Self-Attention の実応用
- 言語理解タスクでの有効性
- 実装細節が詳細

- **特徴**: 最も活発な Transformer 実装

- **使用**: 事前学習モデル + Fine-tuning

```bash
pip install transformers
```

サンプル:
```python
from transformers import AutoTokenizer, AutoModel

tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")

inputs = tokenizer("Hello world", return_tensors="pt")
```

## 📊 インタラクティブなツール

### 1. Transformer Visualization
- **URL**: https://github.com/jessevig/bertviz
- **機能**: Attention ウェイトの可視化
- **用途**: Attention パターンの理解

```python
from transformers import AutoTokenizer, AutoModel
from bertviz import head_view, model_view

model_name = "bert-base-uncased"
model = AutoModel.from_pretrained(model_name, output_attentions=True)
tokenizer = AutoTokenizer.from_pretrained(model_name)

inputs = tokenizer.encode("Hello world", return_tensors='pt')
outputs = model(inputs)
attention = outputs[-1]

head_view(attention, tokenizer.convert_ids_to_tokens(inputs[0]))
```

### 2. Attention Pattern Playground
- **URL**: https://ramasesh.org/patternstm
