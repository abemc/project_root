#### 4. TLS/SSL 証明書管理

**実装:**

```bash
#!/bin/bash
# cert-manager 自動証明書更新

# Helm で cert-manager インストール
helm repo add jetstack https://charts.jetstack.io
helm install cert-manager jetstack/cert-manager \
  --namespace cert-manager \
  --create-namespace

# ClusterIssuer 作成 (Let's Encrypt)
cat <<EOF | kubectl apply -f -
apiVersion: cert-manager.io/v1
kind: ClusterIssuer
metadata:
  name: letsencrypt-prod
spec:
  acme:
    server: https://acme-v02.api.letsencrypt.org/directory
    email: security@company.com
    privateKeySecretRef:
      name: letsencrypt-prod
    solvers:
    - http01:
        ingress:
          class: nginx
EOF

# Ingress で証明書自動取得
cat <<EOF | kubectl apply -f -
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: esp-api
  annotations:
    cert-manager.io/cluster-issuer: "letsencrypt-prod"
spec:
  tls:
  - hosts:
    - api.security-platform.com
    secretName: esp-api-tls
  rules:
  - host: api.security-platform.com
    http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: api-gateway
            port:
              number: 443
EOF
```

**期待効果:**
- 証明書有効期限切れ: 0% (自動更新)
- SSL/TLS 準拠: 100%

#### 5. WAF (Web Application Firewall) 導入

**実装:**

```yaml
# ModSecurity WAF ルール
# /etc/modsecurity/modsec_includes.conf

# OWASP CRS (Core Rule Set)
Include /usr/share/modsecurity-crs/rules/*.conf

# カスタムルール
SecRule REQUEST_URI "@rx ^/admin" \
  "id:1001,phase:2,block,msg:'Admin Access Restricted'"

SecRule REQUEST_HEADERS:User-Agent "@contains sqlmap" \
  "id:1002,phase:2,block,msg:'SQLi Scanner Detected'"

SecRule REQUEST_METHOD "!@rx ^(GET|POST|PUT|DELETE)$" \
  "id:1003,phase:2,block,msg:'Invalid HTTP Method'"

# レート制限
SecRule IP:@countlist "!/login" \
  "phase:5,setvar:ip.login_count=+1,expirevar:ip.login_count=3600"

SecRule IP:login_count "@gt 10" \
  "phase:2,block,msg:'Rate Limit Exceeded'"
```

**期待効果:**
- SQL インジェクション検出: +99%
- XSS 攻撃検出: +98%

#### 6. バックアップ暗号化 & オフサイト保管

**実装:**

```bash
#!/bin/bash
# オフサイト暗号化バックアップスクリプト

# Step 1: DB バックアップ
pg_dump production_db > /tmp/backup.sql

# Step 2: 暗号化 (AES-256)
openssl enc -aes-256-cbc -salt -in /tmp/backup.sql \
  -out /tmp/backup.sql.enc -k "$BACKUP_PASSWORD"

# Step 3: S3 (別リージョン) にアップロード
aws s3 cp /tmp/backup.sql.enc \
  s3://esp-backup-secondary-region/$(date +%Y%m%d_%H%M%S).sql.enc \
  --region us-west-2 \
  --sse AES256

# Step 4: 整合性チェック
openssl enc -aes-256-cbc -d -in /tmp/backup.sql.enc \
  -out /tmp/backup.sql.dec -k "$BACKUP_PASSWORD"

diff /tmp/backup.sql /tmp/backup.sql.dec && echo "✓ Backup Verified"

# Step 5: ローカル削除
shred -vfz /tmp/backup.sql*
```

**期待効果:**
- バックアップ暗号化: 100%
- データ可用性: 99.999% (3地域複製)

#### 8. AI/ML セキュリティ異常検知

**実装:**

```python
# 異常検知モデル
import tensorflow as tf
from sklearn.preprocessing import StandardScaler

class SecurityAnomalyDetector:
    def __init__(self):
        self.model = tf.keras.Sequential([
            tf.keras.layers.Dense(64, activation='relu', input_dim=20),
            tf.keras.layers.Dense(32, activation='relu'),
            tf.keras.layers.Dense(16, activation='relu'),
            tf.keras.layers.Dense(20, activation='sigmoid')
        ])
        self.model.compile(optimizer='adam', loss='mse')
        self.scaler = StandardScaler()

    def train(self, normal_traffic_data):
        """正常なトラフィックで学習"""
        X_scaled = self.scaler.fit_transform(normal_traffic_data)
        self.model.fit(X_scaled, X_scaled, epochs=50, batch_size=32)

    def detect_anomaly(self, traffic_features):
        """異常検知"""
        X_scaled = self.scaler.transform([traffic_features])
        reconstruction = self.model.predict(X_scaled)
        mse = tf.reduce_mean(tf.square(X_scaled - reconstruction)).numpy()
        
        threshold = 0.05
        if mse > threshold:
            return {
                'is_anomaly': True,
                'anomaly_score': mse,
                'severity': 'CRITICAL' if mse > 0.1 else 'HIGH'
            }
        return {'is_anomaly': False, 'anomaly_score': mse}

# 使用例
detector = SecurityAnomalyDetector()
detector.train(normal_traffic_data)
result = detector.detect_anomaly(current_traffic)

if result['is_anomaly']:
    alert_team(f"🚨 異常検知: スコア {result['anomaly_score']:.3f}")
```
