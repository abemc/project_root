## 🚀 クイックスタート

### 1️⃣ 環境セットアップ

```bash
# 仮想環境を有効化
source .venv/bin/activate

# 依存パッケージをインストール
pip install -r requirements.txt
```

### 2️⃣ アプリケーション実行

```bash
# Streamlit UI を起動
streamlit run app.py

# または、メイン処理を実行
python main.py
```

### 3️⃣ デプロイメント

```bash
# デプロイスクリプトを実行
bash tools/deploy.sh
```

## 🔍 主要コンポーネント概要

### Phase 7 ★最新実装
- **QueryPreprocessor**: クエリの高度な前処理
- **KnowledgeIntegrationEngine**: ドメイン知識統合推論
- **ContextAnalyzer**: コンテキスト分析
- **DomainKnowledgeManager**: ドメイン知識管理

### 本番化対応
- **ProductionManager**: リソース・セキュリティ管理
- **ErrorHandler**: 統一エラー処理・リカバリー
- **PerformanceMonitor**: パフォーマンス監視・最適化

### 自己改善・学習
- **FeedbackManager**: フィードバック収集・分析
- **PromptOptimizer**: プロンプト最適化
