## Day 3-5: 新機能開発 (3日)

### プロジェクト3: 信頼度スコア改善

**タスク**: UncertaintyManager を改善して信頼度を22%向上させる

```python
# uncertainty_manager.py の改善例
class ImprovedUncertaintyManager:
    def __init__(self):
        self.confidence_weights = {
            'domain_match': 0.4,
            'temporal_relevance': 0.3,
            'source_reliability': 0.3
        }
    
    def calculate_confidence(self, query, context):
        """改善された信頼度計算"""
        scores = [
            self.domain_match_score(query, context),
            self.temporal_score(context),
            self.source_score(context)
        ]
        return sum(s * w for s, w in zip(
            scores, 
            self.confidence_weights.values()
        ))
```

## Week 5: 統合プロジェクト

### Day 1-3: 統合実装プロジェクト (3日)

**メガプロジェクト: エンドツーエンド推論パイプライン**

```python
# integrated_pipeline.py
class Phase7CompletePipeline:
    def __init__(self):
        self.domain_manager = DomainKnowledgeManager()
        self.knowledge_integrator = KnowledgeIntegrator()
        self.causal_reasoner = CausalReasoningEngine()
        self.uncertainty_mgr = UncertaintyManager()
    
    def process_query(self, query):
        """完全なパイプライン処理"""
        # Step 1: ドメイン推定
        domain = self.domain_manager.infer_domain(query)
        
        # Step 2: 関連知識統合
        integrated = self.knowledge_integrator.integrate(
            domain, query
        )
        
        # Step 3: 因果推論
        causality = self.causal_reasoner.reason(
            query, integrated
        )
        
        # Step 4: 不確実性評価
        confidence = self.uncertainty_mgr.evaluate(
            causality
        )
        
        return {
            "answer": causality.result,
            "domain": domain,
            "confidence": confidence,
            "sources": integrated.sources
        }
```
