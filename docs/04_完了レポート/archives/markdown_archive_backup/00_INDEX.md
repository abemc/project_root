**対象**: 中級者～上級者  
**ボリューム**: 700+ 行  
**実行時間**: 30分

**含まれる実装**:

```python
# 基本関数 (50行)
scaled_dot_product_attention()
    ├─ Q, K, V の処理
    ├─ スコア計算
    └─ softmax + 出力

# クラス実装 (500行)
MultiHeadAttention
    ├─ head 分割
    ├─ 並列計算
    └─ 連結
    
SelfAttention
CrossAttention
CausalAttention

TransformerBlock
    ├─ Attention
    ├─ Feed-Forward
    ├─ Layer Norm
    └─ 残差接続

# ユーティリティ (150行)
可視化 (ヒートマップ)
位置エンコーディング
ベンチマーク
テスト検証
```

**使用方法**:

```python
from attention_utils import scaled_dot_product_attention

# 基本的な使用
output, weights = scaled_dot_product_attention(Q, K, V)

# Multi-Head
attn = MultiHeadAttention(d_model=512, num_heads=8)
output = attn(Q, K, V)

# Transformer ブロック
transformer = TransformerBlock(d_model=512, num_heads=8)
output = transformer(x)
```

**テスト実行**:

```bash

# テストが自動実行され、全ユーティリティの動作確認可
```
