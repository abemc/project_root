### 問題: メモリ不足エラー

**症状**: `MemoryError` 発生

**対応**:
```python
# キャッシュサイズを削減
config.max_cache_size_mb = 200
retriever.clear_cache()
```

### 問題: 高CPU使用率

**症状**: `cpu_percent > 80%`

**対応**:
```python
# ワーカー数削減
config.max_workers = 2
config.resource_constraint = ResourceConstraint.STRICT
```

### 問題: タイムアウト増加

**症状**: `client_timeout` エラー量増加

**対応**:
```python
# タイムアウトを延長
config.query_timeout_sec = 40
config.retrieval_timeout_sec = 30
```

### 問題: セキュリティアラート

**症状**: `blocked_requests` が増加

**対応**:
```python
# レート制限を確認
metrics = manager.security_manager.get_security_metrics()
print(f"ブロック率: {metrics['block_rate']}")

# 必要に応じてリセット
manager.security_manager.request_log.clear()
manager.security_manager.blocked_requests = 0
```
